API Name API Description Parameters
1. Oceans Returns the ocean or sea for given latitude/longitude
Parameters: lat (range -90 - 90), lng (range -180 - 180)
2. Time Zones Returns a list of postalcodes and places for the lat/lng query
Parameters: lat (range -90 - 90), lng (range -180 - 180)
3. Country Code returns the iso country code for the given latitude/longitude
Parameters: lat (range -90 - 90), lng (range -180 - 180)
